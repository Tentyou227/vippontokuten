あと一歩で公開できます

このアプリは無料のリアルタイム置き場「Firebase」とつないで動きます。下の手順は5分ほどです。

  1. Firebase で無料プロジェクトを作成
  2. 「Realtime Database」を作成（ロケーションは任意）
  3. 歯車 → プロジェクトの設定 → アプリ「</>（ウェブ）」を追加し、表示される firebaseConfig をコピー
  4. このHTMLを開いて、const firebaseConfig = {…} の中身を貼り替える
  5. Realtime Databaseのルールを {"rules":{".read":true,".write":true}} にして公開

そのあと、このHTMLを無料の置き場（GitHub Pages / Netlify / Cloudflare Pages）に上げると、配れるURLができます。
